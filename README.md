🚢 Global Shipping & Red Sea Crisis Analytics

📌 Project Overview
This project simulates and analyzes the profound impact of the late-2023 Red Sea shipping crisis on global supply chains. By analyzing 8,000 shipment records over 3 years, this dashboard quantifies the financial and operational toll of rerouting vessels from the Suez Canal to the Cape of Good Hope.

🚀 Key Business Insights
Freight Cost Explosion: Identified a massive spike in average freight costs post-December 2023, directly correlating with the forced route changes.
Operational Delays: Calculated an average delay of 18+ days for shipments forced to take the Cape of Good Hope route, significantly impacting inventory turnover.
Route Shift Analysis: Visualized the stark transition in post-crisis route distribution, with the vast majority of shipments abandoning traditional transit corridors.
Dark-Themed UI: Developed a premium, industrial-themed standalone HTML dashboard utilizing CSS Grid for clear executive reporting.

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Line, Bar, and Donut charts)
UI/UX: Standalone HTML/CSS Grid Dashboard (Dark Mode)
